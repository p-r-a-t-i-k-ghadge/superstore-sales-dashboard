# 📊 Superstore Sales Dashboard

A complete Business Intelligence dashboard built in Google Colab using the Superstore dataset.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/p-r-a-t-i-k-ghadge/superstore-sales-dashboard/blob/main/Superstore_Sales_Dashboard.ipynb)

---

## 📌 Project Overview

This project performs end-to-end sales analysis on the Superstore dataset using Python.

## 📊 Analyses Included

| # | Analysis | Chart Type |
|---|----------|------------|
| 1 | Monthly Revenue Trend | Plotly Line Chart |
| 2 | Sales by Category & Sub-Category | Plotly Grouped Bar |
| 3 | Regional Sales Performance | Plotly Bar + Pie |
| 4 | Top 10 Products by Profit | Plotly Horizontal Bar |
| 5 | Discount vs Profit Correlation | Plotly Scatter + Trendline |
| 6 | Customer Segment Analysis | Plotly Sunburst |
| 7 | Year-over-Year Growth | Plotly Bar + Line |
| 8 | State × Category Profit | Seaborn Heatmap |

## 🛠️ Libraries Used

- **Pandas** — Data cleaning & manipulation
- **Matplotlib / Seaborn** — Static charts
- **Plotly Express** — Interactive charts

## 📂 Dataset

[Superstore Dataset on Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

## 🚀 How to Run

1. Click the **Open in Colab** badge above
2. Upload `Superstore.csv` to the Colab session (Files panel on the left)
3. Run all cells: **Runtime → Run all**
